# Devanshu Sales Performance Dashboard

A comprehensive sales analytics dashboard providing insights into sales performance across different segments, regions, and time periods.

## 📊 Dashboard Overview

This dashboard provides a complete view of sales performance with interactive visualizations and key metrics tracking.

### Key Metrics
- **Total Sales**: $51.80K
- **Total Profit**: $737.77
- **Total Quantity**: 234 units
- **Total Discount**: $11.60

## 📈 Features

### 1. Sales by Segment Analysis
- **Consumer Segment**: 37.5% (19.43K)
- **Corporate Segment**: 33.04% (17.11K) 
- **Home Office Segment**: 29.46% (15.26K)

### 2. Time-Based Sales Tracking
- Monthly sales trends from 2018-2021
- Peak performance identification
- Seasonal pattern analysis

### 3. Product Category Performance
Top performing sub-categories:
- Bookcases: Leading category
- Binders: 7.53K sales
- Phones: 7.42K sales
- Art: 7.04K sales
- Storage: 5.63K sales
- Tables: 4.09K sales
- Appliances: 3.97K sales
- Chairs: 3.26K sales
- Paper: 2.81K sales

### 4. Regional Sales Distribution
- **Central Region**: 41.3% (21.39K)
- **South Region**: 27.95% (14.48K)
- **West Region**: 23.04% (11.94K)
- **East Region**: 7.7% (3.99K)

### 5. Geographic Visualization
- Interactive US map showing sales distribution
- State-wise performance tracking

## 🛠️ Technology Stack

- **Visualization Platform**: Power BI / Tableau
- **Data Source**: Sales database
- **Charts Used**: 
  - Pie Charts for segment and regional analysis
  - Bar Charts for sub-category performance
  - Time Series Charts for trend analysis
  - Geographic Maps for location-based insights

## 📊 Dashboard Components

1. **Segment Performance Pie Chart**: Shows sales distribution across Consumer, Corporate, and Home Office segments
2. **Time Series Analysis**: Monthly sales trends over 4-year period
3. **Category Performance Bar Chart**: Horizontal bar chart showing sub-category wise sales
4. **Geographic Map**: US map with state-wise sales indicators
5. **Regional Distribution**: Donut chart showing regional sales breakdown
6. **Key Metrics Cards**: Summary statistics for quick reference
